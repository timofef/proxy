#Запуск
```
sudo docker build -t proxy . 
sudo docker run -p 8081:8081 -p 8082:8082 --name proxy -t proxy
```
#Использование
На порте 8081 - прокси сервер.
На порте 8082 - сервис для повтора запросов и проверки запросов на наличие XXE уязвимости.
